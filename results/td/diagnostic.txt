=== Interpretation hint ===
  - If 0x17D0 is ALL ZERO on every CPU: the kernel has not allocated a
    feedback table. HFI is not set up by the OS, regardless of firmware.
    Likely fix: kernel is missing CONFIG_INTEL_HFI_THERMAL=y, or the
    intel_hfi module exists but failed to probe. Check dmesg.

  - If 0x17D0 is NONZERO but 0x17D1 bit 0 is clear: HFI is configured
    but disabled. Something disabled it (e.g. power setting, thermal
    policy). Try enabling with msr write (risky; better to find the
    kernel path that would re-enable it).

  - If 0x17D0 and 0x17D1 both look right but 0x17D2 still all zeros:
    ITD classification itself is gated. Check CPUID.06H EDX bit 23 and
    BIOS "Intel Thread Director" / "Hardware Feedback" toggle.
